# 🌡️ Temperature Converter Website

A responsive web app built in **Programming Fundamentals (Semester 2)** to convert Celsius, Fahrenheit, and Kelvin values.

## 🔑 Features
- Convert between Celsius, Fahrenheit, and Kelvin
- Responsive design with HTML, CSS, JavaScript
- Simple, user-friendly interface

## 🛠️ Tech Stack
- HTML, CSS, Python

## 🎯 Objectives
- Practice frontend development basics
- Apply problem-solving in real-world context

## 📚 Academic Context
Developed in **Programming Fundamentals (Semester 2)** at FAST NUCES Karachi.

## 🚀 Getting Started
Open `index.html` in your browser to run the converter.
